Welcome to the Basics of Arduino

Welcome to the basics of Arduino. Before we start, let me introduce you to what Arduino is.

Arduino is an open-source electronics platform consisting of both hardware and software. It is designed to make electronics projects accessible to everyone. Arduino boards can read inputs, such as light on a sensor, a button press, or even a tweet, and turn them into outputs, like activating a motor, lighting up an LED, or posting something online.

The Arduino IDE (Integrated Development Environment) is the software used to write and upload code to the Arduino boards. This combination allows both beginners and professionals to create interactive projects easily. Arduino is popular due to its user-friendly approach and vast community support, offering countless tutorials and resources.

The void setup() Function

The void setup() function runs only once when the program starts. It acts as the program's starting point but does not execute again after the initial setup.

For example, the command pinMode in void setup() tells the program whether a pin is an OUTPUT or an INPUT.

The void loop() Function

The void loop() function repeats commands endlessly. This is where most of your program's logic will reside.
